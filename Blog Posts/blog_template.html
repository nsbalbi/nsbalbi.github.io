The Div Class

To represent each division (or div) in our system, we're going to need a class. This class will contain the div's properties along with methods to render and iterate the system. To visualize a div, we'll draw a rectange defined by two x-y coordinates. 

class Div {
  // Coordinates defining rectangle
  float x1;
  float y1;
  float x2;
  float y2;

  Div(float x1, float y1, float x2, float y2) {
    // Constructor for first div
    this.x1 = x1;
    this.y1 = y1;
    this.x2 = x2;
    this.y2 = y2;
  }

  void iterate() {
    // Do something
  }

  void render() {
    // Draw rectangle
    rect(x1, y1, x2-x1, y2-y1);
  }
}
